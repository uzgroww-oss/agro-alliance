# Social Automation Worker

**Purpose:** Automate social media posting and statistics gathering.

**Inputs:**
- Scheduled post content
- Social platform API credentials
- Blogger social account references

**Outputs:**
- Published social media posts
- Updated subscriber/statistics data
- Engagement metrics

**Flow:**
1. Read scheduled posts from queue
2. Post to configured platforms (Telegram, Instagram, etc.)
3. Collect engagement statistics
4. Update blogger social stats
