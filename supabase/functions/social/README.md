# Social Domain

Functions: `me-socials-add`, `me-socials-delete`, `me-videos-add`, `me-videos-delete`

- Phase: 7

### me-socials-delete
- Unlink a social account
- Phase: 7

### me-videos-add
- Add a video reference
- Phase: 7

### me-videos-delete
