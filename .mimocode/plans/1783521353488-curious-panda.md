2. Google/GitHub bilan kirish
3. **API Keys** → **Create API Key**

2. `worker-ai-categorize` — Gemini API ga wire

4. `worker-ai-seo` — Gemini API ga wire
5. `worker-ai-translate` — Gemini API ga wire
6. `worker-ai-draft` — Gemini API ga wire
7. `_shared/provider.ts` — Gemini provider class qo'shish
8. News source qo'shish (admin panel orqali)
9. Cron job ni trigger qilish

**Tekshirish:** News source qo'shish → pipeline ishlaydi → maqolalar paydo bo'ladi

**Muddat: 1 session**

**Tekshirish:** News page bo'sh holat ko'rsatadi. AI pipeline ishlagandan keyin maqolalar paydo bo'ladi.

### Phase 3: Admin Dashboard — To'liq CRUD

**Muddat: 2-3 session**

1. Home → homepage_sections + featured news + featured bloggers
