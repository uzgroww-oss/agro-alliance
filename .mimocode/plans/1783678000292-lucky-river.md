# YouTube Analytics API Integratsiyasi

## Maqsad
YouTube kanalidan haqiqiy demografik ma'lumotlarni olish:
- Jins taqsimoti (erkak/ayol %)
- Yosh oralig'lari (18-24, 25-34, 35-44, 45+)
- Hududlar (O'zbekiston viloyatlari)

## Qadamlar

### 1. Google Cloud Console
- YouTube Analytics API ni yoqish
- OAuth 2.0 credentials yaratish
- Redirect URI: `https://ckdgprxppmtsgrofvkxd.supabase.co/functions/v1/youtube-oauth-callback`

### 2. Backend (Supabase Edge Functions)
- `youtube-oauth-start` — OAuth flow ni boshlash
- `youtube-oauth-callback` — Google'dan token olish
- `youtube-analytics` — Haqiqiy analytics olish

### 3. Frontend
- YouTube bilan login tugmasi
- Analytics ko'rsatish

### 4. Database
- `youtube_tokens` jadvali — OAuth tokenlarni saqlash
